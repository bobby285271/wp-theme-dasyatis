## WordPress Theme Dasyatis

一个基于 [Automattic](https://automattic.com/) Maywood 的 WordPress 主题（Maywood 本身为 Varia 的子主题）。

* 主色调调整：`#897248` -> `#6D58B3`，`#685636` -> `#7777DB`，`#c4493f` -> `#3083bb`。
* 版式调整：隐藏独立页面标题（`style.css`）：
```
/*Hide title*/
.page .entry-title{
	display:none;
}
```
